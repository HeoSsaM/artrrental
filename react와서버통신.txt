ajax -> 리액트에서 서버와 통신
1) 정확한 규격에 요청
2) GET or POST (데이터 요청방식)

<form action="요청할 URL" method="post">

- Get/Post 방식으로 데이터를 요청 -> 브라우저가 새로고침

- AJAX : 서버에 get/post 요청을 할 때 새로고침 없이 데이터를 주고받을 수 있게 도와주는 간단한 브라우저 기능

--> 새로고침 없이 쇼핑몰 상품을 더 가져오기 가능
--> 새로고침 없이 댓글작성-> 서버전동

Ajax로 데이터를 요청 
-> XMLHttpRequest 옛날 문법
-> fetch() -> 최신문법 
-> axios ->  외부 라이브러리
---> 객체/데이터 <-> json
